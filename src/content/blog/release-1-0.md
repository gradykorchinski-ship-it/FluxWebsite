Today we’re shipping FluxLinux 1.0 — the first public build of a lightweight, modern distribution focused on performance and simplicity. Highlights:

- Fast boot and responsive desktop
- Minimal, audited default packages
- Improved installer with predictable partitioning

Ways to try it: visit the [download page](/os/download/) for ISO images and installation instructions.
