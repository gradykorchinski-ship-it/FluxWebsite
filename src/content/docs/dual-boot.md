# Dual Booting with Windows

## Prerequisites
- Windows installed (UEFI mode recommended).
- Disable **Secure Boot** in BIOS.
- Disable **Fast Startup** in Windows.

## Steps
1. Shrink your Windows partition using Disk Management.
2. Boot FluxLinux USB.
3. In the installer, choose "Install alongside" or manually partition the free space.
