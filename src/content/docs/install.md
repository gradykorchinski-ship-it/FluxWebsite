# Installation Guide

## 1. Download the ISO
Download the latest FluxLinux ISO from the [Download page](/os/download/).

## 2. Flash to USB
Use a tool like Balena Etcher or Rufus to write the ISO to a USB drive (min 4GB).

## 3. Boot from USB
Insert the USB drive into your computer, restart, and enter the boot menu (usually F12, F10, or Del). Select your USB drive.

## 4. Install
Follow the on-screen instructions in the Calamares installer.
